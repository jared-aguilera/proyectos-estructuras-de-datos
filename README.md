### unidad1_sistema_biblioteca
Aplicación para organizar el inventario de una biblioteca que permite registrar libros con sus datos básicos y buscarlos rápidamente para llevar un control ordenado del catálogo.

### unidad2_sistema_reservas
Sistema para la recepción de un hotel que permite apartar habitaciones por fechas específicas, calcular el costo total según los días de estancia y gestionar cancelaciones o cambios de último momento.

### unidad3_sistema_medico
Sistema de gestión de atención médica que optimiza el flujo de pacientes mediante colas de prioridad, permitiendo la atención inmediata de emergencias, el agendamiento de citas futuras y el seguimiento de estadísticas de eficiencia operativa.
